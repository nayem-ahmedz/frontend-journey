Text Formatting

Tags and their Usage
Tag Name Usage
<b> defines bold text, without any extra importance.
<strong> defines text with strong importance
<i> Defines a part of text in an alternate voice or mood, Italic
<em> defines emphasized text
<small> defines smaller text
<mark> defines marked/highlighted text
<del> text that has been deleted from a document, (strike thorugh)
<ins> text that has been inserted into a document, (underline)
<sub> defines subscript text, Example : H2O
<sup> defines superscript text, Example : A2

Tips

  • <i> indicate a technical term, a phrase from another lang, a thought, a ship name, etc
